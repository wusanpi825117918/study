# 复制、移动文件及目录命令

**学习目标**

* 能够使用Linux命令拷贝和移动文件

---

### 1. 复制、移动文件及目录命令的使用

| 命令 | 说明 |
| :--- | :--- |
| cp | 拷贝文件、拷贝目录 |
| mv | 移动文件、移动目录、重命名 |

**cp命令效果图:**

![cp命令](/linux基础命令/imgs/cp.png)

**说明:**

* 拷贝目录需要加上-r选项，-r表示递归拷贝目录及其内容

**mv命令重名名效果图**

![mv命令](/linux基础命令/imgs/rename.png)

### 2. 小结

* cp：拷贝目录需要加上 –r 选项
* mv：移动目录不需要加选项
* mv：命令还可以重命名文件和目录
