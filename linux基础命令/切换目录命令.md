# 切换目录命令

**学习目标**

* 能够使用Linux命令查看目录信息

---

### 1. 切换目录命令的使用

| 命令 | 说明 |
| :--- | :--- |
| cd 目录 | 切换到指定目录 |
| cd ~ | 切换到当前用户的主目录 |
| cd .. | 切换到上一级目录 |
| cd . | 切换到当前目录 |
| cd - | 切换到上一次目录 |

**注意:**

* cd命令切换目录时，这个目录必须存在。
* cd 后面不写目录等价于cd ~
