### 2. grep命令选项的使用

| 命令选项 | 说明 |
| :--- | :--- |
| -i | 忽略大小写 |
| -n | 显示匹配行号 |
| -v | 显示不包含匹配文本的所有行 |

### 3. grep命令结合正则表达式的使用

| :--- | :--- |
| ^ | 以指定字符串开头 |
| $ | 以指定字符串结尾 |
| . | 匹配一个非换行符的字符 |

### 4. 扩展
* ** grep **命令还可以文本搜索管道中的内容，比如: **ls / | grep ‘lib’** 
* 在使用 ** grep ** 命令的时候还可以省略搜索内容的引号，比如: **ls / | grep lib**， **grep hello 1.txt **
